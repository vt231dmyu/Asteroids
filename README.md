## Controls
**Menu:**
- ArrowUp / ArrowDown - Choose Button
- Space - Click Button

**Game:**
- ArrowUp - Go
- ArrowLeft - Turn Left
- ArrowRight - Turn Right
- ArrowDown - HyperSpace
- Space - Shoot
- Escape - Pause Game
